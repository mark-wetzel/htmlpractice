Links

  • Utilizes the anchor tag: <a>
  • The actual link is the value of the href attribute
  • Absolute path (http://www.google.com) or relative to the current page
    • e.g. <a href="some_page_in_dir.html">Click here</a>
  • You can also link to elements with unique IDs on the page using the hashtag character (#). (View source for ToC)

Images

  • Width and height tags used to tell the browser how large the image will be before it's fully loaded (avoid possible page jumping)
  • Alt attribute for accessibility purposes. Useful information embedded for visually impaired users (utilizing screenreaders)
  • Three common formats: JPEG, GIF and PNG
    • JPEG (jay-peg): Mathematical compression. Some image quality lost. Used most commonly for photographs.
    • GIF (peanut-butter): Images with solid colors such as logos and icons. 256 colors. Full transparency for any pixels.
    • PNG (ping): Similiar to GIF but 16 million color support. Alpha (partial) transparency. Unsupported in some older browsers.

Tables

  • Structure tabular data. Not for designing the layout of a page!
  • <table> defines the table
  • <tr> defines a table row and is enclosed in the <table> tag
  • <td> defines a table data cell and is enclosed in the <tr> tag

Inline Styles

Inline styles are placed directly in html tags, like this:

<div style="background-color:green;"> ... <div>

Inline styles are the most specific way to override styling and will override both internal and external styles for that particular element.

They're generally a bad idea as HTML is about semantic meaning, not presentation. It's also very difficult to update the same style over multiple pages when required.
